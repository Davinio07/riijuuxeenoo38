## 11 September

### Vragen / antwoorden

    1. Wat is het doel van de website?

    -> Data inzichtelijk maken voor de gebruikers, niet informeren over de partijen. Dit weten ze allemaal al.


    2. Wat zijn must-haves voor de website?

    -> Paginas moeten binnen 2 seconden geladen zijn.


    3. Is er een idee voor het visueel design?

    -> Design moet beetje "saai" zijn. Hou het proffesioneel.


### Ideas

    -> Vergelijking van partijen 2024/2025 op verschil
    
    -> Rankings van wie het meest gestegen zijn op basis van filters.

    -> Forum lijkt op een chat

    -> Rekening houden met de doelgroep. Dit is voor de real election nerds.

    -> Responsiveness voor het laatste

    -> Could have is meerdere talen, maar is geen must.

    -> Alle paginas MOETEN binnen 2 seconden laden.

    -> Hou rekening met verschillende provincies. Niet overal kan op alles gestemd worden.
